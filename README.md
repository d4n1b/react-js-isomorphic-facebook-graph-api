# ReactJS isomorphic Facebook Graph API

SPA that shows the basic user details of your Facebook account.

Technologies
  - ReactJS
  - Isomorphic Code
  - Facebook Graph API
  - Webpack 2
  - Redux
  - React Router


Run App

  - Clone repository
  - npm i
  - npm run build && npm run server
  - Open URL http://localhost:3001/ in your browser
